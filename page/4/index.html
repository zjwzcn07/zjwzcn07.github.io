A* 寻路

概要

今天接触了下像素级寻路算法，不同于之前acm拓扑图里的最短路算法，游戏里的寻路算法图更大，限制也更丰富。
主要的寻路算法有A*，dijkstra还有些智能算法如蚁群或者通过遗传进行优化等。
在此我们先尝试下A*找找最短路

从左边某空白处走到右边某空白处，中间黑色为障碍物。试寻最短路

算法

其实A*似乎是dijkstra的一个优化，加了估值函数所以更快更好求解
将图片处理后变为0或1矩阵，1位障碍，0可通行
1）现在我们定义每个点有个属性F,G,H
G是从起点走到当前点需要的步数
H是从该点走到终点的估值步数（在此我们取曼哈顿距离）
F = G + H;
2）我们定义两个数组，close和open
close表示该点能走，open表示能走
3）于是乎我们可以像dijkstra一样一层层往外扩
对于障碍物，我们设为close = 1
对于下一个点，如果是close，那么就是不在走
如果是open
    如果没走过，直接上即可
    若没走过，那么如果下一个点的距离（F）会变小，我们就更新他
4）优化：加上优先队列，每次出队的都是F最小的，可以降低一定的时间复杂度

几何拾遗

概要

做了下CF上ProblemSet的几何题，遇到的一些注意点，持续更新中。
下述按过题人数排序

题集

618C: 给一堆二维平面的点，让你找个三角形使得三角形中没有其他点
做法：从某点出发找到最小的边，再找最小面积即可

498A: 给出很多条线将二维平面划分成许多blocks，问从某block上某点走到另一个
block上某点最少需要经过几个block
做法： 对于每条线判断AB两点是否异侧即可

613A: 给出一堆点，让你找个圆心为原点的最小的环能覆盖这些点
做法： 找到离原点最远的点，和最近的点，最近的点不一定是给出的点，有可能是点
与点之间连线与圆心的最短距离

135B: 给你8个点，问你是否可能其中四个组成正方形，另外四个组成矩形
做法： 正方形的话判断四个点到中心的距离是否相等，和中心的连续是否垂直或者
平行，矩形的话判断四个边是否垂直即可

474C: 给你4个点，问你总共每个点要90度旋转几次才有可能构成正方形
做法： 预处理每个点4次的旋转情况

257C: 给出二维平面一堆点，问你从原点开始覆盖这些点的最小角度是多少
做法： 极角排序后求两点之间最大间隔角度theta, 360-theta即可

231D： 从原点为一角有个正方体，每个面上有val，现在给你一点，问你从这个
视角能看到的总的val是多少。
做法： 直接判断该点相对于每个面的位置即可。

576C: 给你大概1e6个点，坐标小于1e6，问你找条路径链接这些点，任意路径即可，但是总长不超过25*10e8
做法： 假如随便连线极其有可能超过限制长度。那么我们可以尽量连短。类似于桶排，
将1e6分成1000个桶，每个桶内按y排序，螺旋式上升连接

464B: 给你8个点，你可以调换单个点内xyz的值，问你8个点是否可能构成立方体。
做法： 类似于正方形的判断也是根据中心距判断是否合法

660D: 给你一堆点，问你能构成多少个平行四边形
做法： 把每对点的距离斜率hash一下即可，ans = sigma[C(n,2)]/2

14C : 给你4个线段，问是否构成矩形
做法： dfs之后判断面积即可

280A: 以原点为中心有个矩形A，逆时针旋转alpha°后变成B，问你A和B的交面积
做法： B旋转后与A的每个边交点求出来再极角排序求面积即可，注意交点的合法性

Project_Eular_407

Description

If we calculate a^2 mod 6 for 0 ≤ a ≤ 5 we get: 0,1,4,3,4,1.

The largest value of a such that a2 ≡ a mod 6 is 4.
Let’s call M(n) the largest value of a < n such that a^2 ≡ a (mod n).
So M(6) = 4.

Find ∑M(n) for 1 ≤ n ≤ 107.

Solution

如果n是素数或者是素数的幂次，显然 M(n) = 1
如果a^2 mod n = a, 那么 a(a-1) = 0 (mod n)
所以可以这样表示 n = uv, a%u = 0, (a-1)%v = 0
令 a = uw, 因为a<n 并且 w < v, uw = 1 (mod v), 则 w = u^-1 (mod v)
所以只要找到互质的uv，然后找到u在v下的逆元w，选择最大uw即可
至于求逆元，由于uv互质，因为欧拉函数可得, u^-1 = u^(fai(v)-1) (mod v)

Code

ANS = 39782849136421
