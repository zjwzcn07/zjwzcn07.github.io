USACO-6.1-A RectangularBarn

题意

一块R*C的地，有p个坏点，问你最大的不包括坏点的矩阵面积是多少？

解

极大化思想

参见国家集训队王知昆文章
设h[i,j]为点(i,j)向上方扩展的最大高度，
l[i,j]为(i,h[i,j])这条线段向左边扩展的最长距离，tl表示点(i,j)向左扩展的最大距离
r[i,j]为(i,h[i,j])向右边扩展的最长距离，tr表示点(i,j)向右扩展的最大距离
则
H[i,j]=(i,j)损坏？0:H[i-1,j]+1
L[i,j]=(i,j)损坏？INF:min(L[i-1,j],tl)
R[i,j]=(i,j)损坏？INF:min(R[i-1,j],tr)

USACO-5.5-Canada TourCanada Tour

题意

N个点，包括一个起点，从西到东排列，存在M条航线
要求你规划旅游路线，从西到东然后从东到西回来，出来起点，每个点只能经过一次
输出经过最多的点数

解

可以将返程路线反向
则就变成两条从起点出发到终点的路线，且中间经过的点不一样
f[i,j]为假定甲到达i，乙到达j，二者经过路总和
f[i,j]=f[j,i]=max{f[i,k]+1}(若k->j存在路线且f[i,k]大于0)

最后只要找到最大的f[i,N]即可

USACO-5.5-Picture

题意

N个矩形贴在二维平面上，有可能有重合覆盖，求所有矩阵组成的集合的周长
数据为N行，每行为矩阵的左下角和右上角坐标

解

每条横边从下到上排列，纵边从左到右排列
每次遇到起始边则该点++，否则--
当该点标记从0变1时，ans++
从1变0时，ans++

代码

struct node
{
    int p,s,e;
    int start;
}H[10000+1],V[10000+1];
int HH[20000+3],VV[20000+3];
bool cmp(node a,node b)
{
    if(a.p==b.p)
    {
        return a.start>b.start;
    }
    else
        return a.p<b.p;;
}
int N,v,h;
int main()
{
    scanf("%d",&N);
    for(int i=0;i<N;i++)
    {
        int x1,y1,x2,y2;
        cin>>x1>>y1>>x2>>y2;
        V[v].p=x1;
        V[v].s=y1;
        V[v].e=y2;
        V[v++].start=1;
        V[v].p=x2;
        V[v].s=y1;
        V[v].e=y2;
        V[v++].start=0;
        H[h].p=y1;
        H[h].s=x1;
        H[h].e=x2;
        H[h++].start=1;
        H[h].p=y2;
        H[h].s=x1;
        H[h].e=x2;
        H[h++].start=0;
    }
    sort(H,H+h,cmp);
    sort(V,V+v,cmp);
    int ans=0;
    for(int i=0;i<h;i++)
    {
        if(H[i].start)
        {
            for(int j=H[i].s;j<H[i].e;j++)
            {
                HH[j+10000]++;
                if(HH[j+10000]==1)
                    ans++;
            }
        }
        else
        {
            for(int j=H[i].s;j<H[i].e;j++)
            {
                HH[j+10000]--;
                if(HH[j+10000]==0)
                    ans++;
            }
        }
        if(V[i].start)
        {
            for(int j=V[i].s;j<V[i].e;j++)
            {
                VV[j+10000]++;
                if(VV[j+10000]==1)
                    ans++;
            }
        }
        else
        {
            for(int j=V[i].s;j<V[i].e;j++)
            {
                VV[j+10000]--;
                if(VV[j+10000]==0)
                    ans++;
            }
        }
    }
    printf("%d\n",ans);
    return 0;
}
