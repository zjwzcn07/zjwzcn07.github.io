USACO-6.1-calfflac

题意

匹配最大回文串

解

Manacher模板，O(2*n)匹配回文串
将原串每个一个字符插入一个特殊字符例如'#'，即将源串拓展为原来两倍
P[i]代表以i为中心最长回文串的右边长度，则i为中心最长回文串为2*P[i]-1
则源串最长为P[i]-1

从左往右计算P[i]，当计算P[i]时，P[j](j<i)已经计算完毕，设mx为之前计算中

最长回文子串的右端点的最大值，位置为id；
如果i < id ，以i为中心的回文串的长度至少和以j为中心的回文串一样
可先令p[i] = min(mx - i,p[2*id-i]);
再一个个匹配
否则说明对于中点为i的回文串没有匹配，则只好一个个匹配
然后更新mx和id

USACO-6.1-A RectangularBarn

题意

一块R*C的地，有p个坏点，问你最大的不包括坏点的矩阵面积是多少？

解

极大化思想

参见国家集训队王知昆文章
设h[i,j]为点(i,j)向上方扩展的最大高度，
l[i,j]为(i,h[i,j])这条线段向左边扩展的最长距离，tl表示点(i,j)向左扩展的最大距离
r[i,j]为(i,h[i,j])向右边扩展的最长距离，tr表示点(i,j)向右扩展的最大距离
则
H[i,j]=(i,j)损坏？0:H[i-1,j]+1
L[i,j]=(i,j)损坏？INF:min(L[i-1,j],tl)
R[i,j]=(i,j)损坏？INF:min(R[i-1,j],tr)

USACO-5.5-Canada TourCanada Tour

题意

N个点，包括一个起点，从西到东排列，存在M条航线
要求你规划旅游路线，从西到东然后从东到西回来，出来起点，每个点只能经过一次
输出经过最多的点数

解

可以将返程路线反向
则就变成两条从起点出发到终点的路线，且中间经过的点不一样
f[i,j]为假定甲到达i，乙到达j，二者经过路总和
f[i,j]=f[j,i]=max{f[i,k]+1}(若k->j存在路线且f[i,k]大于0)

最后只要找到最大的f[i,N]即可

USACO-5.5-Picture

题意

N个矩形贴在二维平面上，有可能有重合覆盖，求所有矩阵组成的集合的周长
数据为N行，每行为矩阵的左下角和右上角坐标

解

每条横边从下到上排列，纵边从左到右排列
每次遇到起始边则该点++，否则--
当该点标记从0变1时，ans++
从1变0时，ans++

代码

struct node
{
    int p,s,e;
    int start;
}H[10000+1],V[10000+1];
int HH[20000+3],VV[20000+3];
bool cmp(node a,node b)
{
    if(a.p==b.p)
    {
        return a.start>b.start;
    }
    else
        return a.p<b.p;;
}
int N,v,h;
int main()
{
    scanf("%d",&N);
    for(int i=0;i<N;i++)
    {
        int x1,y1,x2,y2;
        cin>>x1>>y1>>x2>>y2;
        V[v].p=x1;
        V[v].s=y1;
        V[v].e=y2;
        V[v++].start=1;
        V[v].p=x2;
        V[v].s=y1;
        V[v].e=y2;
        V[v++].start=0;
        H[h].p=y1;
        H[h].s=x1;
        H[h].e=x2;
        H[h++].start=1;
        H[h].p=y2;
        H[h].s=x1;
        H[h].e=x2;
        H[h++].start=0;
    }
    sort(H,H+h,cmp);
    sort(V,V+v,cmp);
    int ans=0;
    for(int i=0;i<h;i++)
    {
        if(H[i].start)
        {
            for(int j=H[i].s;j<H[i].e;j++)
            {
                HH[j+10000]++;
                if(HH[j+10000]==1)
                    ans++;
            }
        }
        else
        {
            for(int j=H[i].s;j<H[i].e;j++)
            {
                HH[j+10000]--;
                if(HH[j+10000]==0)
                    ans++;
            }
        }
        if(V[i].start)
        {
            for(int j=V[i].s;j<V[i].e;j++)
            {
                VV[j+10000]++;
                if(VV[j+10000]==1)
                    ans++;
            }
        }
        else
        {
            for(int j=V[i].s;j<V[i].e;j++)
            {
                VV[j+10000]--;
                if(VV[j+10000]==0)
                    ans++;
            }
        }
    }
    printf("%d\n",ans);
    return 0;
}

USACO-5.5-hidden

题意

一个字符串首尾成环，从哪个位置开始读入的字符串的值最小，如albabla,
有以下表示方法：
albabla、lbablaa、bablaal、ablaalb、blaalba、laalbab、aalbabl
而答案是aalbabl

解

其实就是最小表示法的学习

参见国家集训队周源文章

    可将源串s后面再加一串自己，即S=s+s
    首先i=0，j=2
    若S[i+k]>S[j+k],显然i到i+k都不会是最小表示，从i到j-1也不会是最小表示，
所以i+k+1小于j时，i就会变成j，否则变成i+k+1,而无论如何，j变成i+1；
    反之，j直接变成j+k+1；

USACO-5.3-Network of School

题意

一些学校连入一个电脑网络。那些学校已订立了协议：
每个学校都会给其它的一些学校分发软件（称作“接受学校”）。
注意即使 B 在 A 学校的分发列表中， A 也不一定在 B 学校的列表中。
你要写一个程序计算，根据协议，为了让网络中所有的学校都用上新软件，
必须接受新软件副本的最少学校数目（子任务 A）。更进一步，我们想要
确定通过给任意一个学校发送新软件，这个软件就会分发到网络中的所有
学校。为了完成这个任务，我们可能必须扩展接收学校列表，使其加入新
成员。计算最少需要增加几个扩展，使得不论我们给哪个学校发送新软件，
它都会到达其余所有的学校（子任务 B）。一个扩展就是在一个学校的接
收学校列表中引入一个新成员。

解

任务一让你求有几个图是连通的，任务二让你求加多少条边使得整幅图强连通。
任务一只要从入度为0的顶点dfs标记，ans1++，之后对于没有标记的点无向dfs。
任务二只要计算出度为0的点记为ans2，若ans2==0，则ans2=ans1-1，否则，
ans2=max(ans1,ans2)。
