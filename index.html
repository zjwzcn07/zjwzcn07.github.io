Project_Eular_407

Description

If we calculate a^2 mod 6 for 0 ≤ a ≤ 5 we get: 0,1,4,3,4,1.

The largest value of a such that a2 ≡ a mod 6 is 4.
Let’s call M(n) the largest value of a < n such that a^2 ≡ a (mod n).
So M(6) = 4.

Find ∑M(n) for 1 ≤ n ≤ 107.

Solution

如果n是素数或者是素数的幂次，显然 M(n) = 1
如果a^2 mod n = a, 那么 a(a-1) = 0 (mod n)
所以可以这样表示 n = uv, a%u = 0, (a-1)%v = 0
令 a = uw, 因为a<n 并且 w < v, uw = 1 (mod v), 则 w = u^-1 (mod v)
所以只要找到互质的uv，然后找到u在v下的逆元w，选择最大uw即可
至于求逆元，由于uv互质，因为欧拉函数可得, u^-1 = u^(fai(v)-1) (mod v)

KMP && Extend_KMP

KMP

主要是求next, next[i]是指前i-1个中最长的的前缀和后缀匹配的长度

Extend_KMP

是指extend[i]代表最长的P[i..i+z-1]==S[0…z-1]的z值

HDOJ_2446

题意

每个球如下堆放，有自己的序列，如第二堆的第三个是4，第三堆的第六个是10


数据范围小于2^63
给出序列s，求该球在第几堆，对几行第几个

解

当前堆之前的所有堆球总数=n+2*(n-1)+3*(n-2)...n=(n+1)(n+2)n/6
找出最大的n小于等于s
然后在该堆中该球前几行的求总数=h*(h+1)/2
二分找出n,h即可，注意最大值的取值要小于(2^63)^(1/3)

POJ_2559_Largest Rectangle in a Histogram

题意

不同高度的直方图组成的图中，找到一个最大的矩形。

解

用单调栈维护，若当前值比栈顶值低，则出栈，边出栈边计算每个出栈值的最大面积
若栈为空，则是当前值高*位置索引，然后压栈

USACO-6.5-betsy's Tour

题意

N*N的格子，从左上角走到左下角，且每个格子都要走且仅走一遍

解

dfs剪枝：
1）若走到某边缘时，左右都还没走，则剪枝
2）若前后两个都走，左右两格没走，则剪枝
3）必经之点为：该点只有一个相邻的未经点
   若当前位置有两个必经点，则剪枝

USACO-6.5-All Latin Squares

题意

给一个数字N
要求Latin Square为N*N矩阵即每个1到N的整数在每行每列都出现且出现一次
现规定，第一行必须为123...N
找出有几个符合的矩阵

解

dfs，关键是剪枝
1） 最后一行不用搜，已经确定
2） 每行的第一个可以当做是该行序号即可，如第二行则第一个是2
    因为每行第一个不同都可以由其他行上下调换而成
    所以算出来总数后乘以(N-1)!即可
3)  利用置换群剪枝，如
A    1 2 3 4 5
    2 1 4 5 3 置换圈为(1,2)(3,4,5) 
B    1 2 3 4 5
    2 3 1 5 4 置换圈为(1,2,3)(4,5)

    AB等价，因为可以把A中的1、2换成4、5；3、4、5变成1、2、3
    所以只要置换圈数量和最大置换圈大小一样，则情况相同

USACO-6.3-Electric Fences

题意

二维平面上有一些线段，让你在平面上找一个点，使得该点到各线段距离和最小

解

    枚举肯定超时，可用模拟退火算法来解
    由初始解i和控制参数初值t开始，对当前解重复“产生新解→计算目标函数差→接受或舍弃”的迭代，
    并逐步衰减t值，算法终止时的当前解即为所得近似最优解
    基本步骤：
1）初始化温度T，和起始解状态S
2）对k=1...L做3-6步
3）产生新解S'
4）计算增量Δt′=C(S′)-C(S)，其中C(S)为评价函数
5）若Δt′<0则接受，否则以概率exp(-Δt′/T)概率接受
6)若存在终止条件则输出当前新解结束程序
7）逐步减小T ------逐步减小T的函数选取很关键（个人感觉）

    迭代次数越多，答案越精确，然而时间也越久
